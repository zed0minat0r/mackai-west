# MacKai West

Specialist recruitment for Tax and Finance & Accounting. Single-page corporate site.

**Live**: https://zed0minat0r.github.io/mackai-west/

## Stack
- Static HTML / CSS / JS
- Playfair Display (display) + Inter (body)
- GitHub Pages deploy via `.github/workflows/pages.yml`

## Working on the site
- Edit `style.css`, then run `npx clean-css-cli -o style.min.css style.css` to regenerate the minified file the page references in production.
- Read `CLAUDE.md` before touching anything — it captures the project rules and the verification protocol.
